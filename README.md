## 常见问题解答

**Q: DoorDash能配送酒类吗？**
A: 可以，但得年满21岁，配送时需要出示ID验证。而且不是所有地区都支持，得看你所在位置。

**Q: 可以预约配送吗？**
A: 可以。你可以提前几小时甚至几天预定，指定具体的配送时间。这对于聚会或者特殊场合很有用。

**Q: 配送费为什么有时候特别贵？**
A: 配送费是动态的，取决于距离、时间段、天气和配送需求。高峰期、恶劣天气或者距离远的话，配送费就会涨。

**Q: 如果东西送错了或者漏了怎么办？**
A: 在App里选择该订单，点"Help"然后选"Order Issues"，描述问题后客服会处理。通常会退款或者重新配送。

**Q: 可以给小费吗？必须给吗？**
A: 小费不是强制的，但建议给。Dasher的收入很大一部分来自小费。你可以在下单时给，也可以配送后用现金给。标准是订单金额的15-20%。
